# CSC3060 Project 4: Cache Simulator Assignment

## Project Overview
This project implements a simplified cache simulator to understand the core concepts of cache hierarchy design, including:
- Address mapping (Index, Tag, Offset)
- Cache hits, misses, and replacement policies (LRU, BIP, SRRIP)
- Multi-level cache interactions (L1 & L2)
- Hardware prefetching mechanisms (NextLine, Stride)
